## Привет. Меня зовут Кристина. Я manual QA Engineer.

Есть опыт в тестировании frontend (адаптивная верстка, соответствие макетам), backend (Swagger, тестирование API через Postman) и составлении тестовой документации (чек-листы, тест-кейсы в Qase, баг-репорты в Jira и Trello).

Понимаю клиент-серверную архитектуру, могу написать SQL запросы, умею работать с DevTools, знакома с Browserstack и Charles.

Сейчас в поиске постоянной удаленной работы. Часовой пояс не имеет значения.
